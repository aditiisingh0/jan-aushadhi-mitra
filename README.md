# Jan Aushadhi Mitra

AI-based web application that helps users find generic alternatives 
for branded medicines and locate nearby Jan Aushadhi stores.

## Features
- Medicine alternative finder
- Price comparison
- Savings calculator
- Jan Aushadhi store locator

## Tech Stack
- HTML
- CSS
- JavaScript
- JSON Database
- Google Maps

## How to Run
Open index.html in browser.

## Impact
This project helps people save money on medicines and promotes affordable healthcare using Jan Aushadhi scheme.
